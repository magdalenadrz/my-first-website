HTML Cheatsheet

What exactly is HTML?

‘Hypertext Markup Language’ (HTML) is a front-end programming language used to create webpages by describing their structure. It is the word component or 'backbone' of a website.

Definitions

Hypertext: text that links to other information by clicking on a link. In HTML, it stands for the hyperlinks that a page may contain.

Markup Language: a computer language which uses tags to define the page layout and elements within a document, an example of such a language being HTML.
